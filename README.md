## Structure
```
├── /Game
│   ├── /assets         Contains the game's assets
│   ├── /lib            Contains external libraries
│   └── /src            Contains the game's source code
│
├── Resources           Contains resources for the game that should not be shipped, like raw audio
│
├── Builds              Contains the builds made with makelove
│
└── Root                Root access to the workspace
```
